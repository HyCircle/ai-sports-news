🏈⚾🏎️ spnews

轻量级体育资讯自动化工具 · RSS 抓取 + LLM 聚类 + 中文早报

Python 3.12+ uv feedparser LLM API
  • 支持 NFL 橄榄球 · MLB 棒球 · F1 赛车
  • 时间窗口过滤 + 自动去重 + HTML 清洗
  • 智能事件聚类，重要性分级（high/medium/low）
  • AI 生成结构化中文总结，来源可追溯
本地运行:
uv sync
export LLM_BASE_URL="http://localhost:8070/v1"
uv run spnews
